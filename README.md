# Digital Naadam Universe

## Төслийн зорилго
Naadam Operating System нь Төв цэнгэлдэхийн үзэгч, жуулчин, худалдаачин, ивээн тэтгэгч, зохион байгуулагч, ажилтнуудад зориулсан бүрэн веб суурьтай дижитал дэд бүтэц юм. Олон хүнтэй орчинд чиглүүлэх, захиалга авах, ослын мэдээлэл шийдэх, өгөгдлийг нэгтгэх асуудлыг нэг платформоор шийднэ.

## Гол боломжууд
- Интерактив газрын зураг ба чиглүүлэлт
- Шууд хуваарь болон эвентийн мэдээлэл
- Хоол/худалдаачны захиалга ба дарааллын үзүүлэлт
- Олон хэлт туслах ба жуулчны мэдээлэл
- Аюулгүй байдлын мэдээлэл, ослын төв
- Тоглоомжуулсан аялал ба урамшуулал
- Худалдаачны портал, захиалга/нөөц/аналитик
- Зохион байгуулагчийн команд төв
- Ивээн тэтгэгчийн портал
- Эвентын дараах тайлан, аналитик

## Тех стэк
- Next.js 16 (App Router)
- React 19 + TypeScript
- Tailwind CSS 4

## Локал ажиллуулах
```bash
npm install
npm run dev
```

Хяналтын командууд:
```bash
npm run lint
npm run build
```

## Демо урсгал
1. Ландинг хуудсаас үзэгчийн демо руу орно.
2. Үзэгч газрын зураг ашиглан хоолны бүсийг олно.
3. Хоол захиалж, захиалгын төлөвийг харна.
4. Хуваарьт сануулга хадгална.
5. Асуудал мэдээлээд зохион байгуулагч дээр шууд харагдана.
6. Худалдаачин захиалгаа удирдана.
7. Ивээн тэтгэгч кампани үзүүлэлтээ харна.
8. Зохион байгуулагч тайлан үүсгэнэ.

## Ирээдүйн өргөтгөл
- Бодит цагийн backend API, WebSocket холболт
- Олон хэлний автомат орчуулга, аяллын зөвлөмж
- Бодит төлбөрийн системтэй холболт
- Дэд бүтэц сайжруулалт, олон эвент дэмжих
